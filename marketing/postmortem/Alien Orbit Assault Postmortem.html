The Postmortem

ALIEN ORBIT ASSAULT is a vertical shoot-'em-up in the Toaplan tradition — think Truxton, Hellfire, Out Zone. Browser-based, keyboard or pad, no install. Reclaim Earth from Gaia's corrupted orbit while juggling three weapons, a Substrate economy, and a multi-phase Orbiter boss whose 30-second vulnerability window you have to earn down.

The brief was small enough to fit on one line: a Toaplan-feel shmup I could ship in a month, in the browser, by myself. The interesting bit isn't the game — it's that I built it without ever opening a sprite editor, a DAW, or really fighting the engine. The whole stack is AI-native. The whole feel is mine.

The Loop

The spine is a single shootable pickup, the Cycler: it drifts down the playfield through four armed states — vulcan, shotgun, beam, bonus substrate. Shoot it to advance, touch it to commit. Miss the timing and it locks to the consolation state. That one pickup replaced an inventory of four colored loot drops on the first back-and-forth riff with Claude.

The Workflow

Heavy lifting — multi-file refactors, the Orbiter phase machine, the input Action / Binding layer — went to Claude Code. Light edits and boilerplate stayed in VS Code with Copilot on auto, which keeps the bill honest on one-line fixes.

Art was a single fictional magazine cover from Nano Banana; its palette, mood, and ship silhouette anchored every later sprite. Music was three loops from Suno — title, run, boss — picked by ear at cab volume. LittleJS sits underneath: small enough for the model to hold the whole engine in its head, which makes Claude Code dangerously productive against it.

Where the Human Lives

Bullet spread. Ship i-frames. Substrate drop curve. How fast the Orbiter's window should drain on a clean hit. The git history is a long chain of 0.18 → 0.19 → 0.185 nudges. Each commit looks trivial; together they are the game.

Still needed me

Feel Is Not Generated

Bullet spread, ship i-frames, substrate drop curve, the bleed rate on the Orbiter's 30-second window, screen shake on Latch Beam release.

The git log is a long parade of single-number nudges — tweak shotgun spread, then 0.18 → 0.19, then 0.19 → 0.185. Each commit is trivial. Together they're the whole game.

The trick that worked

Faking 3D Tilt with One Sprite

Only the upright ship is in the build. Tilt is faked at draw time: split the sprite down the centerline, squish + darken the half diving into the page, stretch the opposite half. Mirror horizontally for the other direction.
